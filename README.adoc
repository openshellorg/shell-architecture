= shell-architecture
:toc:

Open Shell Organization notes on **shell architecture**: structured data pipelines, display vs transport, and how that relates to SOS and Nushell.

This is a standards / explanation repository—not a product runtime.

== Stance

* **Do not reinvent Nushell.** Endorse it as the cross-platform reference for structured pipelines. Optionally contribute upstream or build companion libraries (`nu-require`).
* PowerShell pioneered object streams; we still **recommend Nushell** for OpenShellOrg / Dev-Centr defaults (syntax and trust model).
* SOS (flag syntax + certification in `open-shell-org`) is a **complementary layer**, not a competing shell.
